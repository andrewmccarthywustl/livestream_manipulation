Available Effects:

--- Basic Effects ---
original - Original image with no modifications
grayscale - Convert image to grayscale
edge - Detect and highlight edges
cartoon - Apply cartoon-like effect
blur - Apply Gaussian blur
sharpen - Sharpen the image
emboss - Apply emboss effect
sepia - Apply sepia tone
negative - Invert image colors
pixelate - Pixelate the image

--- Advanced Effects ---
neon - Apply neon glow effect
vaporwave - Apply vaporwave aesthetic
glitch - Apply digital glitch effect
thermal - Apply thermal camera effect
mirror - Mirror the image horizontally

--- Wild Effects ---
cubism - Create cubism-inspired effect with geometric shapes
dream - Create a dreamy, ethereal effect
hologram - Create sci-fi hologram effect
painterly - Create an artistic, painterly effect
cellular - Apply cellular automaton rules to create evolving patterns
solarize - Create psychedelic solarization effect
kaleidoscope - Create kaleidoscope effect
rgb_scatter - RGB channels scatter in different directions
rgb_directional - RGB shift with channels moving in specific directions
rgb_shift_extreme - Extreme RGB channel shifting effect
neontrails - Create colorful neon trails that follow motion
neontrails - Create colorful gray trails that follow motion
